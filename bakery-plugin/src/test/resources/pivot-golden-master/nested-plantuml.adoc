== Architecture du pipeline

Le pipeline de deploiement comprend plusieurs etapes.

=== Etapes de build

* Preparation
** Installer les dependances
*** Verifier les pre-requis systeme
** Configurer l'environnement
* Compilation
** Compiler le code source
** Executer les tests unitaires
* Empaquetage
** Generer l'artefact
** Signer le binaire
