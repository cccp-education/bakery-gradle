== Résumé

US-CR-V5-7 (3 pts) — 4 refactors qualitatifs sur 4 fichiers.

=== 1. ConfigResolver.resolveAll() data-driven

*Problème* : 9 blocs `resolveDomain` répétitifs (firebase, googleForms, firebaseAuth, comments, analytics, newsletter, theme, layout, language, supportedLanguages).
*Solution* : Restructuré en liste de `DomainResolver<T>` avec `fold` sur `ResolvedConfigs.copy()`.
*Fichier* : `ConfigResolver.kt` — `resolveAll()` passe de 10 appels individuels à une liste data-driven.

=== 2. BfsProximityCache distances → constantes nommées

*Problème* : Magic numbers 1.0/0.7/0.4/0.1/0.0 dans `computeProximity()`.
*Solution* : `companion object` avec `PROXIMITY_SELF`, `PROXIMITY_DIRECT_NEIGHBOR`, `PROXIMITY_DISTANCE_2`, `PROXIMITY_DISTANCE_3_PLUS`, `PROXIMITY_UNREACHABLE`.
*Fichier* : `BfsProximityCache.kt` — 5 constantes nommées, 9 tests existants PASS.

=== 3. GenerateSiteFromIntentionTask.buildSiteYml() → yamlMapper

*Problème* : Concaténation string manuelle `buildString { appendLine(...) }` pour construire le YAML.
*Solution* : `yamlMapper.writeValueAsString(mapOf(...))` + header commentaire + templates commentaire.
*Fichier* : `GenerateSiteFromIntentionTask.kt` — import `yamlMapper`, `buildSiteYml()` réécrit.

=== 4. GenerateArticleTask.buildArticleContent() texte FR → article.lang

*Problème* : Texte "Article généré par l'assistant IA bakery-gradle." hardcodé en français.
*Solution* : `articleGeneratedNote(lang)` avec `when(lang)` → FR/EN/fallback.
*Fichier* : `GenerateArticleTask.kt` — `buildArticleContent(article, lang)`, nouvelle méthode privée `articleGeneratedNote()`.
