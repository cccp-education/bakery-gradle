== Objectif

== Réalisation

=== Baby Step 1 : DDD — SecretField.kt
- `SecretField` sealed class (Password, ApiKey, Token)
- `maskSecret()` fonction pure :
  - `Password` → `***` (masquage total)
  - `ApiKey` → `${value.take(4)}***${value.takeLast(4)}` (préfixe+suffixe visibles)
  - `Token` → `***[${value.length} chars]` (longueur uniquement)
  - Blank → `(not set)`
