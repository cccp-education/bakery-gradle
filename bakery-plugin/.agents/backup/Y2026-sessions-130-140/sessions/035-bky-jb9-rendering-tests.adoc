== Leçons

1. **Thymeleaf Rendering Tests sont un filet anti-bug essentiel** : 5 bugs
   (3 SpEL/OGNL + 1 CSS inline + 1 Elvis) n'auraient jamais été détectés
   par les tests `readText().contains()`.
2. **`and` vs `&&`** : Thymeleaf StandardDialect utilise OGNL, pas SpEL.
   L'opérateur logique est `&&`, pas `and`.
3. **CSS inline `/*[[${}]]*/`** : Le natural template Thymeleaf place la
   valeur dans un commentaire HTML/CSS que le navigateur ignore. Préférer
   `th:inline="javascript"` + `setProperty()` pour les CSS custom properties.
4. **Toujours déclarer les dépendances via `libs.versions.toml`** : pas de
   versions hardcodées dans `build.gradle.kts`.
