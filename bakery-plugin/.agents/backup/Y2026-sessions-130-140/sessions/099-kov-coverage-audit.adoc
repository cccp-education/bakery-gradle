=== Note sur les closures doLast

Les closures `doLast` des TaskRegistrars ne sont pas couvertes par Kover car celui-ci n'instrumente que le sourceSet `main` et `functionalTest`. Or les closures Gradle sont exécutées dans le contexte build du TestKit (functionalTest/Cucumber), pas en JUnit5. Ces ~1000 instructions comptabilisées comme "non couvertes" le sont en réalité via les FT et Cucumber — Kover ne les voit juste pas.
