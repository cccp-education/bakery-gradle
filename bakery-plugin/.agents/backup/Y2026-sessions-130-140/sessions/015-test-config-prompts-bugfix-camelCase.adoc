=== Bug fix — camelCase → UPPER_SNAKE casse

* [x] `ConfigPrompts.kt:31` : `.uppercase()` etait appele **avant** `.replace(Regex, ...)`
  → plus aucune minuscule, la regex ne matche jamais

  → les variables d'environnement n'etaient jamais resolues correctement
* [x] Correction : inverser l'ordre → `.replace...uppercase()`
