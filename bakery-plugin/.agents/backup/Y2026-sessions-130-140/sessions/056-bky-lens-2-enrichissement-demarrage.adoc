== Architecture LENS-2 (en cours)

```
composite-context.json (runner-gradle N3)
    ↓ AugmentedContextResolver.resolve()  ← NOUVEAU
    ├── EAGER : gouvernance déterministe (.adoc)
    ├── RAG   : similarité vectorielle (pgvector)
    ├── GRAPHIFY : relations structurelles (graphe)
    └── DOCS  : corpus documentaire (codex pgvector)
    ↓ AugmentedArticlesService.score()   ← À IMPLÉMENTER
    site-graph enrichi (~200 nœuds, ~500 edges)
```

== Scoring hybride LENS-2 (formule)

```
score = α × ragSimilarité      // 0.40
      + β × graphProximité     // 0.30
      + γ × tagRecouvrement   // 0.15
      + δ × crossRefCount     // 0.15

  + bonus crossRefBonus (0.2) si prioritizeCrossReferences
  + bonus communityAffinity (0.3) si même communauté
```
