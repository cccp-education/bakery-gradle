## 1. Positionnement — Expert Métier JBake

Bakery est l'expert métier du **site statique JBake** dans l'ecosystème.
Son domaine : scaffolding, templates Thymeleaf, theming, services agrégés, déploiement gh-pages.

```
DOMAINE BAKERY (JBake) :
├── scaffolding site     → generateSite, site.yml, structure JBake
├── templates Thymeleaf  → blog.thyme, page.thyme, post.thyme, menu.thyme, etc.
├── theming              → CSS, variables, layouts, dark/light mode, responsive
├── services agrégés     → Firebase, Disqus, Google Forms, Pagefind ✅, Analytics
├── contenu              → articles, pages, taxonomies, feed.xml, sitemap.xml
└── déploiement          → bake, deploySite, gh-pages, sous-domaines
```

HORS domaine :
- ❌ Pédagogie/formation → c'est training-gradle
- ❌ Slides → c'est slider
- ❌ Documents PDF → c'est codex
- ❌ Quiz → c'est forms/quizz-benchmark
- ❌ IA → c'est codebase (bakery consomme)

## 4. Contrat d'entrée — formation.yaml (optionnel)

bakery expose un contrat optionnel `formation.yaml` que les autres briques (training-gradle, runner) peuvent déposer pour structurer un site formation :

```yaml
# deposé dans le site JBake par training-gradle / runner
formation:
  title: "Titre Formation"
  sessions:
    - id: s01
      title: "Introduction"
      modules:
        - id: m01
          title: "Concepts clés"
          slides: slides/m01.html
          pdf: pdfs/m01.pdf
          quiz: quizzes/m01.html
          google-form-id: "abc123"
```

Ce contrat est **optionnel** — bakery fonctionne sans. Il sert uniquement quand formation il y a.

## 5. IA — Architecture

bakery → codebase :

```
BKY-JB-8 (bakery)
  │
  ├── 1. Requête : "génère article intro session 1"
  │
  ▼
codebase (koog + langchain4j)
  │
  ├── 2. buildContext : RAG pgvector → AFNOR/REAC
  ├── 3. generate : LLM pool gpt-oss-12b:cloud
  ├── 4. selfReflect : auto-évaluation score
  └── 5. Return : article AsciiDoc structuré
      │
      ▼
bakery
  └── 6. Injecte dans JBake : content/s01-intro/index.adoc
  └── 7. bake + deploySite
```

BKY-IA-0 (LlmService, OllamaLlmService) est à dépiler — bakery ne fait plus d'IA directe.
