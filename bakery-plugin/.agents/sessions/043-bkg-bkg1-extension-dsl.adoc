== Leçons apprises
. **OGNL vs SpEL** : L'opérateur Elvis `?:` n'est pas supporté en OGNL (Thymeleaf standard mode). Utiliser `${var != null ? var : 'default'}` à la place.
. **Anti-pattern post-bake** : Injecter du HTML dans les pages baked via Jsoup est un anti-pattern. L'architecture correcte est : config DSL → jbake.properties → Thymeleaf côté serveur.
. **Jsoup supprimé** : Plus aucune dépendance jsoup dans le projet — tout le rendering passe par Thymeleaf.
