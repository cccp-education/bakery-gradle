= Session 065 — BKY-IA-1 Scaffolding assisté par IA (Étapes 1-3)

== Date : 2026-06-03

== Résumé

Dépilage BKY-IA-1 (Scaffolding assisté par IA) — baby steps DDD/TDD/BDD étapes 1 à 3.
Nouveau package `bakery/scaffold/` avec modèle domaine, générateur IA, tâche Gradle, DSL.

== Étapes réalisées

=== Étape 1 — DDD : Modèle domaine (17 UT PASS)

* `ScaffoldIntention` data class (description, siteType, lang, projectName) avec validation + `toPromptContext()`
* `ScaffoldSiteType` enum (BLOG, PORTFOLIO, DOC, FORMATION) avec `fromStringOrDefault()`
* `ScaffoldOutput` data class (siteType, projectName, description, templates, metadata)
* `ScaffoldMetadata` data class (title, description, tags, layout, language)
* `ScaffoldIntentionDsl` bridge Gradle → domaine avec `toIntention()`
* Tests : `ScaffoldIntentionTest` (13 UT) + `ScaffoldIntentionDslTest` (5 UT)

=== Étape 2 — TDD : ScaffoldGenerator (18 UT PASS)

* `ScaffoldGenerator` : prompt LLM structuré + parse JSON réponse + fallback résilient
* Prompt demande JSON avec siteType, projectName, description, templates, metadata
* Parse JSON maison (sans Jackson) — `extractJson`, `extractJsonValue`, `extractJsonArray`
* Fallback par type : BLOG → blog.thyme/post.thyme/page.thyme, DOC/FORMATION/PORTFOLIO templates dédiés
* Slugify description → projectName fallback
* Tests : `ScaffoldGeneratorTest` (17 UT + compact JSON test)

=== Étape 3 — TDD : GenerateSiteFromIntentionTask + DSL wiring (13 UT PASS)

* `GenerateSiteFromIntentionTask` — tâche Gradle `@DisableCachingByDefault`
  * Résolution CLI > DSL > defaults (même pattern que `GenerateArticleTask`)
  * Propriétés : scaffoldDescription, siteType, scaffoldLang, projectName
  * `resolveIntention()` fusionne les sources
  * `applyScaffoldOutput()` écrit `site.yml` suggéré par l'IA
* `BakeryExtension` : nouveau bloc `bakery { scaffoldIntention { ... } }` + action method
* `SiteManager.registerGenerateSiteFromIntentionTask()` — même pattern que `registerGenerateArticleTask()`
* `BakeryPlugin` : branchement dans `afterEvaluate` branche scaffolding
* Tests : `GenerateSiteFromIntentionTaskTest` (3 nested classes, 10 UT)
