Le `auditDir` configuré via `layout.buildDirectory.dir(...)` n'est pas résolu correctement quand le projet de test n'a pas de build réel.

*Solution proposée prochaine session* :
. Soit retirer `dependsOn(bake)` et laisser l'utilisateur exécuter `bake` avant.
. Soit mocker/préparer un répertoire `build/bake` avec du HTML dans les steps Cucumber (déjà fait via `createBakedHtmlWithInlineContrast`, mais le `auditDir` n'est pas résolu vers ce chemin car le registrar l'écrase).
