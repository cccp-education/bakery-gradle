== Décisions techniques

. *DTO intermédiaire pour YAML* : Jackson `@JsonTypeInfo` sur sealed
  interface Kotlin non honoré (tag YAML natif `!<site>` au lieu de
  `type: site`). Solution : `SiteNodeDto` sealed avec annotations + mappers
  `toDto()`/`toDomain()`. `USE_NATIVE_TYPE_ID` disabled sur YAMLFactory.
. *`@JsonIgnore` sur content* : l'AST PivotArticle n'est pas persisted dans
  `site.yml` — seul le path est sérialisé. `content` est restauré au runtime
  par le parser AsciiDoc (TREE-7 scaffold IA).
. *BakeryWorld.siteTree* : ajout d'un champ partagé pour les steps Cucumber
  TREE-2 (pattern existant respecté — injection `BakeryWorld`).
. *Anglais imposé* : utilisateur a demandé nommage/commentaires/Gherkin en
  anglais. Correction appliquée a posteriori sur les 4 features + 4 steps.
