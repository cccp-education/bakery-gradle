== Critère de classification

* **SINGLE-SHOT** : le test crée un projet frais, lance Gradle une seule fois, assert sur l'output. Aucune dépendance sur un état précédent. `withDebug(true)` safe.
* **MULTI-SHOT** : le test lance Gradle plusieurs fois séquentiellement (ex. `bake` x2 pour incremental), partage un projet via `@BeforeAll`, ou effectue des mutations du filesystem entre invocations Gradle. `withDebug(true)` risqué (classloader stale, état résiduel).
