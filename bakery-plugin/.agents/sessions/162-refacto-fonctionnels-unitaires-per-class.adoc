== Objectif

Suite à l'investigation BKY-PERF (profiling : 4min10s pour 39 functional tests,
bottleneck incompressible `GradleRunner.create().build()` forke un daemon à chaque
appel), migration des tests DSL triviaux (compilation + enregistrement tâche)
vers des tests unitaires ProjectBuilder in-process avec `@TestInstance(PER_CLASS)`
et champs `project`/`ext` partagés via `@BeforeEach`.

=== BKY-PERF-6 — Migration FT→UT avec @TestInstance(PER_CLASS) (cette session)

Migration de 6 classes de test fonctionnelles vers des unitaires :

. **IaFunctionalTests** (4 FT) → **IaConfigUnitTest** (4 UT) — `bakery/llm/` — configureIa() helper
. **ArticleIntentionFunctionalTests** (4 FT) → **ArticleIntentionUnitTest** (4 UT) — `bakery/article/` — configureArticleIntention() helper
. **ThemeIntentionFunctionalTests** (5 FT) → **ThemeIntentionUnitTest** (4 UT) — `bakery/theme/` — configureThemeIntention() helper
. **ScaffoldIntentionFunctionalTests** (5 FT) → **ScaffoldIntentionUnitTest** (5 UT) — `bakery/scaffold/` — configureScaffoldIntention() helper
. **CollectSiteContextFunctionalTest** (4 FT nested) → **CollectSiteContextUnitTest** (4 UT) — `bakery/lens/` — siteYamlContent property
. **ValidateFirebaseConfigFunctionalTests** (2 DSL tests supprimés) → **FirebaseAuthConfigUnitTest** (3 UT) — `bakery/` — inline DSL

Pattern commun (6 classes) :
- `@TestInstance(TestInstance.Lifecycle.PER_CLASS)` — 1 instance pour toute la classe
- `@TempDir lateinit var projectDir: File` — shared temp dir
- `private lateinit var project: Project` — created in `@BeforeEach setUp()`
- `private lateinit var ext: BakeryExtension` — created in `@BeforeEach setUp()`
- `@BeforeEach fun setUp()` — `ProjectBuilder.builder().withProjectDir(projectDir).build()`
  → `project.pluginManager.apply(BakeryPlugin::class.java)`
  → `ext = project.extensions.getByType(BakeryExtension::class.java)`
  → `ext.configPath.set(projectDir.resolve("site.yml").absolutePath)`
- Helpers privés configure*() configurent le DSL + appellent `(project as ProjectInternal).evaluate()`
- `assertThat(ext.*.*)` au lieu de `assertThat(result.output).contains("BUILD SUCCESSFUL")`

Tests qui nécessitent un vrai `GradleRunner` (génération fichiers réelle, injection
propriétés) conservés dans **IntegrationRestoredFunctionalTests** (inner classes :
ThemePresetInjectionTest, ScaffoldExecutionTest, CollectGroupVisibilityTest).
