== Leçons apprises

1. **DuplicateStepDefinition** — Déclarer un step `@Then("the file ... should contain ...")` dans 2 fichiers Cucumber crée une DuplicateStepDefinitionException qui empêche TOUS les steps d'être enregistrés. Solution : réutiliser les steps partagés.

2. **Cucumber DSL insertion** — Insérer un bloc DSL dans `build.gradle.kts` via `lastIndexOf('}')` nécessite un `\n` avant le bloc inséré, sinon la syntaxe Kotlin est invalide (deux éléments sur la même ligne).

3. **Cucumber Background** — Les features BKY-IA-2 fonctionnent avec la branche scaffolding (`does not have 'site.yml'`), pas la branche site configuré. `generateSite` injecte les propriétés thème via `ConfigResolver` dans les deux branches.
