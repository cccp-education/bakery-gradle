== Contrat metadata.json

```json
{
  "source": "bakery",
  "type": "site",
  "version": "1.0",
  "model": "jbake",
  "generatedAt": "2026-05-30T06:59:32Z",
  "sessions": 4,
  "dependencies": ["jbake"],
  "articles": [
    {"title": "Fourth Post", "url": "/blog/2013/fourth-post.html", "date": "2013-10-17", "tags": [], "author": ""}
  ],
  "sitemap": ["/about.html", "/blog.html", ...],
  "feed": "/feed.xml"
}
```

Format aligné sur Metadata(codex, training, planner, quizz) — `source`, `type`, `version`, `model`, `generatedAt`, `sessions`, `dependencies`.
