== Objectif

BKG-1.4 — Modifier `related-articles.thyme` pour consommer le graphe d'articles connexes (lookup par slug dans `relatedArticlesData` JSON injecté via jbake.properties) et affichage via JavaScript côté client.

== Tâches réalisées

=== Étape 1 — TDD : Template + injection relatedArticlesData

* [x] `related-articles.thyme` — Remplacement de la boucle Thymeleaf `${related_suggestions}` (non fonctionnelle en contexte JBake) par :
  - `th:data-related-articles="${relatedArticlesData}"` → JSON inline du graphe
  - `th:data-max-results="${relatedArticlesMaxResults}"` → nombre max de suggestions
  - JavaScript `JSON.parse(aside.dataset.relatedArticles)` → lookup par `content.uri` → affichage dynamique
  - Fallback : si pas de suggestions pour l'URI courante → `aside.style.display = 'none'`
  - `th:if="${relatedArticlesEnabled != null && relatedArticlesEnabled == 'true'}"` conservé (guard)
* [x] `SiteManager.injectRelatedArticlesConfigIntoJbakeProperties()` enrichi :
  - Lecture du fichier graphe `graphFilePath` → injection de `relatedArticlesData` (JSON string échappé) dans jbake.properties
  - Fallback : si le fichier n'existe pas → `relatedArticlesData=` (vide)
* [x] `RelatedArticlesTemplateTest` — +2 tests BKG-1.4 (data-related-articles avec JSON / sans données)
* [x] `RelatedArticlesInjectionTest` — +2 tests RelatedArticlesDataInjection (graph file exists / doesn't exist) + méthode enrichie

== Architecture BKG-1.4

```
related-articles.json (collectRelatedArticles)
        ↓
SiteManager.injectRelatedArticlesConfigIntoJbakeProperties()
        ↓ lit le fichier graphe → injecte relatedArticlesData (JSON string)
jbake.properties:
  relatedArticlesEnabled=true
  relatedArticlesMaxResults=4
  relatedArticlesHeading=Articles connexes
  relatedArticlesData={"suggestions":{...},"blogArticles":{...}}
        ↓
related-articles.thyme:
  th:if guard (enabled)
  th:data-related-articles (JSON)
  th:data-max-results (int)
  JavaScript: JSON.parse → lookup content.uri → affiche title + url
  Fallback: pas de suggestions → aside caché
```
