== Apprentissage

. La tâche doit être enregistrée **avant** le branchement `if (!configFile.exists())` pour être accessible dans les deux modes.
. `@DisableCachingByDefault` est requis sur les tâches de validation de fichiers externes pour éviter les faux positifs en cache.
. Le masquage des secrets dans les logs est une responsabilité de la tâche elle-même, pas du plugin — cela garantit la sécurité même en mode `--info`.
. `ProjectBuilder` + `@TempDir` permettent de tester la tâche sans toucher au `site.yml` réel qui contient des secrets.
