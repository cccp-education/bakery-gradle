== Réalisations

=== Commit parser JBake natif (dc249f3)

Parser AsciiDoc détecte et parse le format JBake natif (`= Title / @Author / :jbake-*:`)
en plus du format pivot (`title=.../~~~~~~`). 5 UT + 2 real-file UT + 3 Cucumber BDD
(feature 49). Bump BOM 0.0.9, Gradle 9.6.1, api-key-pool 0.0.1, i18n-contracts 0.0.2.

=== Prompt LLM amélioré (5574132)

`LlmServiceTranslationAdapter.buildPrompt` réécrit : rôle "professional translator",
interdiction explicite de bavardage/options/alternatives. Avant le LLM renvoyait des
méta-commentaires ("Here are a few ways to translate this..."). Maintenant il renvoie
une traduction propre. 5 TDD tests.

=== JbakeNativeRenderer (60fd685)

Nouveau renderer préservant le format JBake natif pendant la traduction.
`PivotFrontmatter` étendu avec `author`, `jbakeAttributes`, `asciidocAttributes`.
`ContentTranslationService` détecte le format source et sélectionne le bon renderer.
5 UT + 3 BDD unit tests + 3 Cucumber scenarios (feature 50).
Interface `ArticleRenderer` extraite pour polymorphisme.

=== Parallélisation traduction (a13c212, 659939f, a0fe81b)

`ContentTranslationService` accepte `parallelism` (default=1). `parallelism>1` utilise
`Dispatchers.IO` + `Semaphore` pour traduire N fichiers concurrents. 4 TDD tests :
équivalence séquentielle, détection concurrence, collecte erreurs, comportement par défaut.
Câblage DSL : `ContentMigrationIntentionDsl.parallelism` → Intention → Task → Service.
Dogfood config : `parallelism = 4`.

=== Exclusion paths dans traduction (8305f05)

`ContentTranslationService.translate()` accepte `excludeRelativePaths` pour skipper
les répertoires exclus pendant la traduction (pas seulement la copie). Corrige le bug
des fichiers `draft/` traduits malgré l'exclusion.

=== Sélecteur de langue dans menu.thyme

Dropdown "Langues" ajouté à `office/sites/cheroliv.com/jbake/templates/menu.thyme`
(10 langues : FR, EN, AR, BN, ES, HI, PT, RU, UR, ZH). Golden master + fixture +
translations_en régénérés pour les nouvelles clés `menu.11`, `menu.12`.

=== Traduction 106/109 articles EN

106 articles traduits en anglais via LLM (`gemma4:31b-cloud` port 11441, parallélisme ×4).
Format JBake natif préservé. 3 fichiers edge case (`:jbake-title:` sans `= Title`)
non traduits — à corriger dans une prochaine session.

=== Déploiement cheroliv.com

Site FR + EN baked et déployé sur `cheroliv.github.io` (commit `1a09c03`).
EN accessible dans `/en/`. Sélecteur de langue visible sur le site live.
